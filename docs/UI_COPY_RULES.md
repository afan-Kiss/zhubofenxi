# UI 文案规则

经营 BI 看板面向老板查看经营情况，文案应 **温和、清晰、可理解**，避免考核、归责、财务术语。

---

## 禁用词（用户可见文案）

以下词汇 **不得** 出现在主页面、Drawer、按钮、提示条等用户可见位置：

| 类别 | 禁用词 |
|------|--------|
| 排名/竞争 | TOP、排名、贡献 |
| 客户标签 | 黑名单 |
| 人事/财务 | 提成、工资、扣款 |
| 归责 | 责任、归责 |
| 财务模块 | 毛利润、平台结算、账单对账、财务中心 |

> 说明：`docs/` 目录可在文档中 **列举** 禁用词本身；源码扫描见 `npm run check:copy`。

---

## 推荐替代表达

| 场景 | 推荐说法 |
|------|----------|
| 主播页 | 主播表现、主播情况 |
| 客户页 | 客户情况、复购客户、高价值客户 |
| 风险提示 | 建议重点关注、建议谨慎发货 |
| 总称 | 经营数据 |
| 金额 | 支付金额、退款金额 |
| 售后 | 品退情况、商品问题售后 |
| 复购 | 复购客户 |

## 页面级示例

### 经营总览

- ✅ 「本期支付金额」「退款金额」「品退情况」
- ❌ 「GMV 贡献 TOP」「毛利润」「平台结算」

### 主播业绩

- ✅ 「主播表现」「品退率（本月）」
- ❌ 「业绩排名」「贡献榜」「提成基数」

### 买家排行

- ✅ 「消费情况」「建议重点关注客户」
- ❌ 「黑名单客户」「退款 TOP10」

### 系统设置

- ✅ 「数据同步」「直播号配置」「重建买家排行」
- ❌ 「财务对账」「工资核算」

## 金额与数字

- 金额：`¥10,079.90`（完整，禁止「万」缩写）
- 数量：`1,234`（可用千分位，禁止「1.2万单」）
- 比率：`3.27%`；分母无效时显示 `--`

---

## 多直播号

- 大盘标题/卡片：**不**逐号拆分（合并展示）
- Drawer 列：**来源直播号**（或「订单来源直播号」）
